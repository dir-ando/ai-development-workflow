# 準備

GitHub repositoryを準備して、Claude Code Actionを呼び出せる状態を作るのが最初の目的です。

次図の流れで進んでいきます。

1. このGitHub Repositoryをfork
2. GitHub Pagesの有効化
3. Claude GitHub Appのインストール
4. Repository SecretにAnthropic API KEYの設定
5. Claude Codeが動くか実験
