# 概要

本研修は、GitHub Actionsで動作するClaude Codeを活用し、開発プロセスの自動化と効率化を図るための実践的なアプローチを体験することが目的です。
従来の開発手法に生成AIの力を活用することで、開発速度の向上、コード品質の担保、そしてエンジニアの生産性向上を実現します。

ソフトウェア開発の知識は必要ありません。

この研修では、人間の手によって1文字もコードを書かないことを目指しています。

ソフトウェアエンジニアの方もそうでない方も、生成AIを活用して自身のアイデアを形にする体験ができます。

ソフトウェア開発者の方には開発フローに生成AIを組み込んで活用する例と具体的なコードサンプルを提供します。

# 利用する技術要素

- GitHub Actions
- GitHub Pages
- Claude Code
- Next.js

## GitHub Actions

GitHubから生成AIを呼び出す際に利用します。
GitHub Actionsから[Claude Code Action](https://code.claude.com/docs/ja/github-actions)を呼び出すことで、Anthropic社のClaudeモデルを呼び出します。

このClaudeモデルを活用してアプリケーションを開発したり、コードのレビューや修正を実施します。

## GitHub Pages

コードだけだと開発できてるかわからないので、実際にどのような画面が出来上がっているのかを確認するために利用します。

個人で利用する場合、GitHub repositoryはPublicであることが必要です。

## Claude Code

Anthropic社が開発しているClaudeモデルをコマンドラインから利用できるように開発されているソフトウェアです。

ChatGPTやGeminiなどのWebインターフェースは誰でも活用できて便利ですが、よりエンジニアリングに活用しやすくするために開発されました。

この研修では、Claude CodeをGitHub Actionsから呼び出します。
Webブラウザではない場所から生成AIを呼び出せるということがソフトウェア開発にもたらす恩恵を感じていただければと思います。

## Next.js

この研修においては重要ではありません。

2026年現在、Webアプリケーションを開発するために比較的良く使われているもの、程度の理解で十分です。
